PressBook Masonry Blogs is a simple, modern, and perfect theme with a masonry grid layout.

== Description ==

A unique and multipurpose theme designed for professional bloggers, news writers, educational websites, and portfolios. PressBook Masonry Blogs comes with a masonry grid layout for blog posts listing available with multiple options for customization like the number of columns, column margin, etc. The branding area of the header includes the logo and title at the center. This theme is beautifully designed and well-crafted for a clean, refreshing, and user-friendly look. The theme is fully mobile responsive, SEO-optimized, lightweight and fast with minimal dependency. Also, this theme supports infinite scrolling and load more button with the Jetpack plugin as well as WooCommerce for creating an eCommerce online shop. This theme can also be used by freelancers, digital marketing agency, story writers, businesses, IT firms, job publishers, events, recruiters, corporates, affiliates, consulting institutes, schools, and academic sites. The features include a multi-level accessible dropdown menu, multiple RGBA and gradient color options, related posts grid, single and double sticky sidebars, top navigation and social links menu, sidebar and footer widgets, and many more. This theme can also be the best choice for photobloggers, vloggers, movies, images and videos gallery, gaming, news media, magazines, and entertainment websites. Also, it is a perfect choice for creative graphics designers, artists, photography, and publishers. With multiple page templates and page-level settings, the theme is compatible with block editor as well as page builder plugins like Elementor, WPBakery, etc. The theme comes with many advertisement spaces including the top banner with a link option. It is easy to set up and you can quickly create an eye-catching, simple, and elegant blog with modern typography.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

PressBook Masonry Blogs includes support for WooCommerce and for Infinite Scroll in Jetpack.

= Where to find the theme options? =

All of the options can be found in the Customizer in 'Appearance > Customize'.

= How to configure the theme widgets? =

The theme widgets can be found in 'Appearance > Widgets'. You can drag and drop widgets to the widget area. PressBook Masonry Blogs comes with 6 widgets area locations which include a left sidebar, right sidebar widgets area and 4 locations for footer widgets.

= How to add the menus or social links? =

The menus can be found in 'Appearance > Menus'. First, create the menu and then assign it to a menu location. PressBook Masonry Blogs comes with three menu locations which include the primary menu, top bar menu, and social links menu. For social links, create a menu item with a custom link, enter the URL of the social page, and assign this menu to the 'Social Links Menu' location.

= How to change the number of blog posts in the grid? =

You can change the number of blog posts from your WordPress admin 'Settings > Reading'. Here, you can change the 'Blog pages show at most' option from 10 to any other value like 12.

= How to add infinite scroll or load more button for the blog posts? =

You can install and activate 'Jetpack' plugin. Then, go to 'Jetpack' > 'Settings' > 'Writing' > 'Infinite Scroll' to add infinite scroll or load more button for the blog posts.

= How to set custom exceprt or summary for the post? =

You can set a custom excerpt or summary of a post on the right side of the post editor screen > 'Post' > 'Excerpt' field.
